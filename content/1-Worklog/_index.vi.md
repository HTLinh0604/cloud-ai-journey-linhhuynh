Đây là nhật ký công việc của mình trong suốt chương trình thực tập **First Cloud AI Journey (FCAJ)** — chương trình thực tập tập trung vào Cloud và AI trên nền tảng AWS. Mỗi nhật ký tuần ghi lại mục tiêu, các công việc triển khai từ Thứ Hai đến Thứ Sáu, và kết quả đạt được.

Chương trình kéo dài **12 tuần** (20/04 – 12/07/2026), bao gồm các nội dung từ cơ bản đến nâng cao về điện toán đám mây, data engineering, analytics, bảo mật và kiến trúc hệ thống trên AWS.

---

| Tuần | Khoảng thời gian | Nội dung tập trung |
|------|-----------------|-------------------|
| [**Tuần 1**](1.1-Week1/) | 20/4 – 26/4 | Thiết lập tài khoản, nhận credits, hoàn thành 5 task thực hành nền tảng, IAM, quản lý chi phí, S3 & EC2 |
| [**Tuần 2**](1.2-Week2/) | 27/4 – 3/5 | Cấu hình S3 lifecycle, quản lý EBS volumes & snapshots, tự động hóa với EBS DLM, thiết lập RDS |
| [**Tuần 3**](1.3-Week3/) | 4/5 – 10/5 | Sử dụng AWS CLI, DynamoDB & NoSQL, CloudFront & Lambda@Edge, thiết lập custom VPC với EC2 Web Server, Lambda serverless, CDK & VS Code Toolkit |
| [**Tuần 4**](1.4-Week4/) | 11/5 – 17/5 | Triển khai IaC với CloudFormation, AWS Glue ETL & Data Catalog, Athena SQL, cấu hình RDS PostgreSQL/MySQL trong VPC |
| [**Tuần 5**](1.5-Week5/) | 18/5 – 24/5 | Thành thạo AWS CLI (EC2/S3/IAM), sync dữ liệu lên S3, tối ưu chi phí S3, backup EBS với DLM, lập kế hoạch DR |
| [**Tuần 6**](1.6-Week6/) | 25/5 – 31/5 | DynamoDB Streams & Global Tables, cache với DAX, quản lý RDS Multi-AZ & backup, so sánh SQL vs NoSQL |
| [**Tuần 7**](1.7-Week7/) | 1/6 – 7/6 | Quy trình ETL & analytics pipeline, xây dựng data lake (S3+Glue+Athena), CDK & CloudFormation nâng cao, serverless workflow |
| [**Tuần 8**](1.8-Week8/) | 8/6 – 14/6 | Mô phỏng cục bộ với LocalStack & Floci, nghiên cứu kiến trúc Data Pipeline hiện đại, thiết kế sơ đồ pipeline |
| [**Tuần 9**](1.9-Week9/) | 15/6 – 21/6 | Thiết kế kiến trúc workshop, tối ưu chi phí dịch vụ, best practices về scalability & security, đánh giá trade-off |
| [**Tuần 10**](1.10-Week10/) | 22/6 – 28/6 | Nghiên cứu sâu Data Engineering, sự kiện AWS tại văn phòng Amazon, phân chia module project, lập roadmap triển khai |
| [**Tuần 11**](1.11-Week11/) | 29/6 – 5/7 | Triển khai data pipeline bằng Python, AWS CLI và Terraform, provisioning hạ tầng, kiểm thử tích hợp E2E |
| [**Tuần 12**](1.12-Week12/) | 6/7 – 12/7 | Hoàn thiện Streamlit dashboard, kiểm thử và debug, tối ưu hiệu năng, viết tài liệu kỹ thuật, push GitHub |
