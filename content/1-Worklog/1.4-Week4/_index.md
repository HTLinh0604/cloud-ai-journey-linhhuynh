Weekly objectives:
- Deepen knowledge of AWS CloudFormation for infrastructure automation.
- Understand AWS Glue components: Data Catalog, Crawlers, and ETL jobs.
- Query S3 datasets using Amazon Athena serverless SQL.
- Implement security best practices in virtual private cloud environments.

Tasks to be deployed this week:

| Day | Task | Date |
|---|---|---|
| Monday | Study CloudFormation stack sets, nested stacks, change sets, and drift detection. | May 11 |

| Thursday | Provision Amazon RDS PostgreSQL instances in a custom VPC with security group chaining. | May 14 |
| Friday | Configure VPC flow logs, Network Access Control Lists, and PrivateLink endpoints. | May 15 |

Weekly results achieved:
- Built reusable, nested infrastructure templates with AWS CloudFormation.
- Generated dynamic metadata schemas automatically using Glue Crawlers.
- Managed query execution costs by establishing Athena workgroup scan thresholds.
- Enforced zero-trust networking by implementing strict security group boundaries for database instances.
