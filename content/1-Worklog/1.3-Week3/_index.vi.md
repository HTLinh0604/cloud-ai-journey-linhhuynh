Mục tiêu tuần:
- Học AWS CLI để quản lý tài nguyên và lọc truy vấn.
- Tìm hiểu các khái niệm cơ sở dữ liệu NoSQL và Amazon DynamoDB.
- Hiểu về Amazon CloudFront và Lambda@Edge cho xử lý tại biên.
- Triển khai cơ sở dữ liệu và ứng dụng web trong VPC tùy chỉnh.
- Thực hành tính toán serverless và Cơ sở hạ tầng dưới dạng mã.

Các công việc cần triển khai trong tuần này:

| Thứ | Công việc | Ngày |
|---|---|---|
| Thứ Hai | Thực hành cơ bản AWS CLI, các thao tác bảng DynamoDB, cấu hình CloudFront và triển khai web server EC2 trong VPC tùy chỉnh. (Lên văn phòng) | 4/5 |
| Thứ Ba | Học về tính toán serverless với AWS Lambda và định nghĩa Cơ sở hạ tầng dưới dạng mã sử dụng AWS CDK trong VS Code. (Lên văn phòng) | 5/5 |
| Thứ Tư | Học sâu về các template AWS CloudFormation và thực hành triển khai RDS PostgreSQL trong VPC tùy chỉnh. (Lên văn phòng) | 6/5 |
| Thứ Năm | Ôn tập các lệnh CLI và thực hành quản lý tệp S3 qua giao diện dòng lệnh. | 7/5 |
| Thứ Sáu | Học các pattern L2 construct của AWS CDK và tìm hiểu về CloudFormation Change Sets. | 8/5 |

Kết quả đạt được trong tuần là gì:
- Sử dụng thành thạo AWS CLI để kiểm tra và lọc các tài nguyên AWS.
- Triển khai mạng VPC tùy chỉnh với subnet công cộng và riêng tư, EC2 instance và cơ sở dữ liệu RDS.
- Tạo và triển khai thành công các ứng dụng serverless sử dụng AWS Lambda.
- Xây dựng hạ tầng tự động bằng AWS CDK và các template CloudFormation.
